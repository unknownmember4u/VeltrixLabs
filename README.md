## Overview

STELLER STATE is a full-stack industrial IoT orchestrator built for HackByte 4.0.
It simulates a smart factory with real-time monitoring, AI-driven anomaly detection,
predictive simulation, and autonomous supply chain management.

---

## Features

- Real-time IoT monitoring
- AI-powered diagnostics (RAG + Ollama)
- Energy dashboard and load balancing
- Simulation lab with 3D models
- Supply chain automation
- Cryptographic audit logs

---

## Tech Stack

- Next.js, React, TailwindCSS
- Three.js (3D rendering)
- SpacetimeDB (Rust WASM)
- Flask (Python backend)
- Ollama + ChromaDB (AI)
